# SyllabiQ-service Docs

This folder contains architecture notes, database schema and Mermaid diagrams for the backend service.

Files:
- `ARCHITECTURE.md` — short architecture overview
- `SCHEMA.md` — table & field summary
- `ER_DIAGRAM.mmd` — Mermaid ER diagram
- `SEQUENCE_DIAGRAMS.mmd` — Mermaid sequence diagrams for main flows

How to view diagrams:
- GitHub renders Mermaid in `.mmd` files. In VSCode, install the "Mermaid Preview" extension and open the `.mmd` file.
- You can also paste the Mermaid blocks into https://mermaid.live to preview/export images.
